# CLI Weather App

## ☁️ Installation

```sh
# Using npm
npm i wet-cli
```

## 📝 How to start usage ?

```sh
# Set token
weather -t [API_KEY]

# Or

# Use standard token
weather -st

# Set city
weather -s [CITY]
```

### Нou can ask for help on command:

```sh
weather -h
```
